## 🎮 controls

### 📋 menu controls
- **Triangle**: Exit to XMB
- **D-Pad**: Navigation
## episode menu controls
- **R**: Next episode
- **L**: Previous episode
- **CIRCLE**: Previous menu
- **CROSS**: Select
- **SQUARE**: Manage game saves

### 🎬 gameplay controls
1. **During a cutscene**:
   - Press **R** and **L** in turn: Return to the menu without saving.
   - Press **R** and **START** in turn: Save and return to the menu.
